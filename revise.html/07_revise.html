Web Development Glossary

HTML
HyperText Markup Language – the standard language for creating web pages and web applications.
CSS
Cascading Style Sheets – used to describe the look and formatting of a website written in HTML.
JavaScript
A programming language used to create dynamic and interactive effects within web browsers.
Frontend
The part of a website that users see and interact with – built using HTML, CSS, and JavaScript.
Backend
The server-side of a website that handles data, storage, and business logic – often built with databases and server languages.
